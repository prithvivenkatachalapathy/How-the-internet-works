About


This website is made as a part of an individual assignment for the class IS 601, Spring 2021 semester taught by professor Keith Williams.

Purpose of this assignment is to get familiarized with how the Internet works, this website goes over five different topics to better understand how internet works. The topics are as below:

  • Domain names and ICANN
  • IP Addresses, Packets and Routing
  • DNS
  • HTTP and HTTPS protocols
  • W3C, HTML and CSS
