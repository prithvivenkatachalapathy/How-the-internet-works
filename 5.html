W3C, HTML and CSS

The World Wide Web Consortium (W3C)

W3C is the “World Wide Web Consortium” founded in 1994 and currently led by Tim Berners-Lee. It is an international community where members, organization and the public works together to develop the web standards. The Web standards are defined for an Open Web Platform for application development that has the unprecedented potential to enable developers to build rich interactive experiences. Which are created by various languages used to code and develop webpages. Standards developed by the Consortium include:

  1. CGI
  2. CSS
  3. DOM
  4. HTML
  5. HTTP
  6. XHTML
  7. XML

HTML

HTML stands for Hypertext Markup Language and was invented by Tim Berners-Lee. The HTML is a markup language that is really straightforward and easy to learn even for complete beginners in website building. It is the language for describing the structure of the web pages. It allows the user to create and structure sections, paragraphs, headings, links, and blockquotes for web pages and applications.

CSS

Cascading Style Sheets (CSS) is a style sheet language used for describing the presentation of a document written in a markup language such as HTML. CSS is used to add colors, create layouts, set various types of fonts to make the data look more presentable on the choice of the developer and designer, so that the webpage looks more appealing and presentable. However, HTML works extremely well with two other frontend languages: CSS (Cascading Style Sheets), and JavaScript. Together, they can achieve rich user experience and implement advanced functions.
  • CSS is responsible for stylings such as background, colors, layouts, spacing, and animations.
  • JavaScript lets you add dynamic functionality such as sliders, pop-ups, and photo galleries
